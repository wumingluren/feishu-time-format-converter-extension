# 飞书多维表格插件-时间格式转换

# 介绍

可以自由转换文本字段类型里面的时间内容为自定义的时间格式。

支持单个单元格、整个字段进行转换，非时间内容可以自动跳过。

# 使用方法

1. 选择模式和范围
2. 设置转换后的格式
3. 点击【执行转换】按钮即可
